# ESFL Fork of Quivr

This repository is a fork of the Quivr project, created for further development by ESFL. The original Quivr repository is no longer being actively maintained, so this fork will serve as the base for our continued development.

## Branch Information

- **Branch Name**: `esfl-fork`
- **Based On**: Quivr v0.0.322
- **Created On**: 2023-11-15

## Current Additions

- **PROMPTS.md**: Documentation explaining how prompts are created and used in Quivr, with a focus on how user instructions are stored in the database and how they are incorporated into the prompts.

## Development Guidelines

### Getting Started

1. Clone this repository:
   ```
   git clone <repository-url>
   git checkout esfl-fork
   ```

2. Follow the installation instructions in the original README.md to set up the development environment.

### Making Changes

1. Create feature branches from the `esfl-fork` branch:
   ```
   git checkout -b feature/your-feature-name esfl-fork
   ```

2. Make your changes and commit them with descriptive messages.

3. Push your changes to the remote repository:
   ```
   git push origin feature/your-feature-name
   ```

4. Create a pull request to merge your changes into the `esfl-fork` branch.

### Code Standards

- Follow the existing code style and patterns in the codebase.
- Write tests for new features and bug fixes.
- Update documentation as needed.

## Future Development

This fork will be used to:

1. Fix bugs in the existing codebase
2. Improve the prompt system to better incorporate user instructions
3. Add new features as needed
4. Keep dependencies up to date
